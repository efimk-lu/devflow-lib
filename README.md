A demo nodeJS client library used to demonstrate CI/CD principles. You can use it as a template.

# Installation

`npm install devflow-client`

# Using

```javascript 1.7
let addTwoNumbers = require(devflow - lib);
console.log(addTwoNumbers(x, y));
```
